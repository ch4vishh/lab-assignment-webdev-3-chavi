This project is a simple and responsive personal portfolio website created using HTML and CSS. It includes multiple sections such as Hero, About, Skills, Projects, and Contact. The design is clean, modern, and mobile-friendly with the help of Flexbox, Grid, animations, and media queries.

Features

✔ Responsive Layout

The website automatically adjusts for desktop, tablet, and mobile screens using CSS media queries.

✔ Sticky Navigation Bar

A fixed blue navbar stays at the top while scrolling for easier navigation.

✔ Modern Hero Section

Includes a big heading, animated text (“Full Stack Developer”), and a profile image placed using Flexbox.

✔ Skills Section with Grid

Skills are displayed in a 3-column CSS Grid layout, each styled as a card with hover effects.

✔ Project Cards with Links

Projects are shown as interactive cards, each linking to the GitHub repository with short descriptions.

✔ Contact Form

A simple form with Name, Email, and Message fields styled with shadows and rounded corners.

✔ Smooth UI Effects

Hover effects, transitions, color animations, and clean spacing improve the user experience.

✔ Clean, Easy-to-Read Code

The project follows a clear structure and uses proper indentation, comments, and modern CSS techniques.
